## Instalación
````bash
composer install
````

## Configuraciones y requerimientos

### PHP
* Version 7.4+
* Configuración
    Configurar el ``php.ini`` para permitir la subida de imagens en caso de requerir subir imagenes de alta calidad
    ````bash    
    upload_max_filesize   100M
    post_max_size         100M
    memory_limit          256M
    ````
